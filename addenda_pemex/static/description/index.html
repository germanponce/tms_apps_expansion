Configuracion.

1. El primer punto es instalar los módulos necesarios para poder configurar y trabajar con Addendas dentro de Odoo versión 12, para ello es necesario instalar los módulos:

  • l10n_mx_einvoice_addenda_no_sale
  • addenda_pemex

2. Al instalar el modulo de addenda_pemex se genera una Plantilla de Addenda y se agregan los siguientes campos dentro del formulario de Clientes:

  • Clave del Acreedor
  • Clave de transportista
  • ID Analítico
  • Analítico
  • No. Cedula
  • Contrato SIIC

3. Al Seleccionar la Plantilla PEMEX automáticamente se estará insertando un fragmento del XML con los datos para su generación, los cuales al crear Facturas para este cliente se estarán reemplazando con los datos necesarios, agilizando el proceso Electrónico.

4. Al momento de crear una Factura para el cliente con la Plantilla de Addenda PEMEX, se habilitaran los campos de la generación de Addendas.

5. Continuando con el formulario de captura de la Factura, se habilitara una pestaña llamada CFDI Addenda.

Los unicos valores adicionales y necesarios son:
  • Año Ejercicio
  • Año Relación
