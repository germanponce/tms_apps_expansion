Acuerdos de Compra - Secuencias Dinamicas

Este modulo agrega las siguientes caracteristicas:

  • Permite tener Secuencias Dinamicas en Acuerdos de Compra.

Funcionamiento

Se agrega un tipo de acuerdo dentro del formulario el cual nos permite generar una secuencia individual por cada uno de estos tipos:

  • Estandar
  • Mantenimiento
  • Insumos


Cada uno de ellos tiene una secuencia configurable y editable dentro del menu Configuración -> Ajustes -> Secuencias de Identificadores -> Secuencias

Secuencia Estandar
  • Secuencia común de los Acuerdos.




Secuencia Mantenimiento
  • Secuencia relacionada con el registro de secuencia "purchase.requisition.mro"




Secuencia Insumos
  • Secuencia relacionada con el registro de secuencia "purchase.requisition.supplies"
